Font Style

CSS dapat mengatur style teks dari yang ada pada halaman web yang dibuat. Terdapat beberapa property yang dapat digunkana untuk style teks yaitu sebagai berikut.

  • text-align: digunakan untuk mengetur perataan teks, value yang dapat digunakan yaitu left, center, right
  • color: digunakan untuk meberi warna pada teks
  • letter-spacing: mengatur jarak antara karakter
  • word-spacing: mengatur jarak antar kata
